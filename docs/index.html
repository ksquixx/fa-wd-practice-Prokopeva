Задание 15

    Это первый абзац с отступом в начале строки. Команда &nbsp; создает неразрывный пробел, который используется для формирования красной строки. В HTML обычные пробелы игнорируются браузером, если их несколько подряд, поэтому для создания видимого отступа нужно использовать специальные символы.

    Это второй абзац, также начинающийся с отступа. Неразрывные пробелы полезны не только для формирования красной строки, но и для предотвращения переноса слов в нежелательных местах. Например, если нужно, чтобы "10 км" или "г-н Иванов" всегда отображались на одной строке, следует использовать &nbsp; между словами.

План приготовления кофе

  1. Нагреть воду до 90-96°C
  2. Смолоть кофейные зерна
  3. Положить фильтр в воронку
  4. Залить кофе горячей водой
  5. Дать настояться 3-4 минуты

Ингредиенты для салата

  • Помидоры - 2 штуки
  • Огурцы - 1 штука
  • Лук репчатый - 1 головка
  • Зелень (петрушка, укроп)
  • Растительное масло для заправки

Задание 21

Термины из веб-разработки:

HTML
HyperText Markup Language - язык разметки, используемый для создания структуры веб-страниц. Определяет элементы страницы, такие как заголовки, абзацы, ссылки и изображения.
CSS
Cascading Style Sheets - язык стилей, описывающий внешний вид HTML-документов. Позволяет задавать цвета, шрифты, расположение элементов и адаптивный дизайн.
JavaScript
Высокоуровневый язык программирования, который добавляет интерактивность веб-страницам. Позволяет создавать динамически обновляемый контент, анимации и сложные веб-приложения.
API
Application Programming Interface - набор протоколов и инструментов для создания программного обеспечения. Определяет способ взаимодействия различных компонентов программы между собой.
Фреймворк
Программная платформа, определяющая структуру программной системы. Облегчает разработку, предоставляя готовые решения для типовых задач и стандартизируя подход к написанию кода.
